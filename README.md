# NayePankh Volunteer Management System

A full-stack web application developed for managing volunteer registrations of NayePankh Foundation.

## 🌐 Live Demo

Frontend:
https://nayepankh-volunteer-system-app.netlify.app

Backend API:
https://nayepankh-backend-fzmz.onrender.com


## 🚀 Features

- Volunteer Registration System
- Admin Login Authentication
- JWT Based Authentication
- Admin Dashboard
- View All Volunteers
- Delete Volunteer Records
- Generate CSV Reports
- Generate PDF Reports
- MongoDB Database Integration


## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Deployment
- Netlify (Frontend)
- Render (Backend)
